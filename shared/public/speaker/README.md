# Shared Speaker Assets

Assets in this folder belong to the repository owner and can be reused by any
deck.

Current canonical assets:

```text
shared/public/speaker/linkedin-qr.svg
```

Reference shared assets from `data/person.js` using Vite `?url` imports. Keep
deck-specific copies only when a presentation needs a different QR, profile
photo, badge, or event-specific graphic.
