# Speaker Assets

Shared speaker assets are canonical at:

```text
shared/public/speaker/
```

The owner metadata is canonical at:

```text
data/person.js
```

This deck keeps `decks/github-enterprise-platform/data/speaker.js` only for
talk-specific overrides such as `talkRole`.

Use this folder only when the deck needs a speaker asset that should not be
shared with other presentations.

Optional deck-specific public paths:

```text
decks/github-enterprise-platform/public/speaker/linkedin-qr.png
decks/github-enterprise-platform/public/speaker/profile.png
```
